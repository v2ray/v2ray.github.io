Blackhole

  • Name: Schwarzes Loch
  • Typ: Ausgehend

Blackhole ist ein Protokoll für ausgehende Verbindungen. Es blockiert alle Verbindungen mit vordefinierten Antworten. Kombiniert mit Routing Dies kann zum Sperren des Zugriffs auf einige Websites verwendet werden.

ConfigurationObject

{
  "response": {
    "type": "none"
  }
}

response: ResponseObject

Pre-defined response. Blockhole will send (if any) pre-defined data immediately for any connection passed to it and close the connection.

ResponseObject

{
  "type": "none"
}

type: "none" | "http"

Type of the response, available options are:

  • "none": Default value. Empty response.
  • "http": A valid HTTP 403 response.
